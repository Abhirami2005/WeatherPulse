# WeatherPulse – Intelligent Real-Time Weather Analytics Dashboard


## live link
 "https://abhirami2005.github.io/WeatherPulse/"


## Overview

WeatherPulse is a modern, responsive, and data-driven weather analytics dashboard designed to provide accurate real-time weather insights and forecast intelligence. Built using React, TypeScript, and Tailwind CSS, the application delivers a seamless user experience through dynamic visualizations, location-based weather tracking, and interactive forecasting capabilities.

The platform enables users to monitor current atmospheric conditions, analyze weather trends, access multi-day forecasts, and receive weather-related alerts through an elegant and highly responsive interface.

## Key Features

### Real-Time Weather Monitoring

* Live weather data retrieval
* Current temperature and atmospheric conditions
* Wind speed, humidity, and precipitation metrics

### Smart Location Services

* Automatic GPS-based location detection
* Global city search functionality
* Reverse geocoding support

### Forecast Intelligence

* Multi-day weather forecasting
* Detailed daily weather insights
* Interactive forecast visualization

### User Experience Enhancements

* Fully responsive design
* Dark mode interface
* Smooth animations using Framer Motion
* Search history management
* Dynamic weather-themed backgrounds

### Alert & Notification System

* Severe weather detection
* Browser-based weather notifications
* High-wind and precipitation alerts

## System Architecture

The application follows a component-based architecture that separates business logic, weather services, user interface components, and utility modules to ensure scalability, maintainability, and performance optimization.

## Installation & Setup

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Production Version

```bash
npm run build
```
